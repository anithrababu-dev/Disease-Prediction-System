# 🩺 Disease Prediction System Using Machine Learning

## 📌 1. Objective
The main objective of this project is to develop a machine learning system that predicts possible diseases based on user-provided symptoms.  
It aims to assist in early diagnosis and provide decision support in healthcare.

---

## ❗ 2. Problem Statement
Many people delay consulting doctors due to lack of awareness or accessibility.  
This system helps by providing a preliminary prediction of diseases using symptoms, enabling faster medical attention.

---

## 📊 3. Dataset Used
The project uses the following datasets:
- `dataset.csv` – Contains symptoms and corresponding diseases  
- `Symptom-severity.csv` – Contains weight/importance of each symptom  

---

## 💻 4. Technologies Used
- **Programming Language:** Python  
- **Libraries:**
  - Pandas (data handling)
  - NumPy (numerical operations)
  - Scikit-learn (machine learning algorithms)
- **Tools:** Jupyter Notebook  

---

## ⚙️ 5. Methodology

### Step 1: Data Collection
- Load dataset using Pandas  
- Understand symptoms and disease mapping  

### Step 2: Data Preprocessing
- Handle missing values  
- Convert categorical symptoms into numerical format  
- Normalize/weight symptoms using severity  

### Step 3: Model Building
- Apply machine learning algorithms such as:
  - Decision Tree  
  - Naive Bayes  
  - Random Forest (if used)  

### Step 4: Training the Model
- Split dataset into training and testing sets  
- Train model using training data  

### Step 5: Prediction
- User inputs symptoms  
- Model predicts the most probable disease  

---

## 🏗️ 6. System Architecture
1. **Input:** User enters symptoms  
2. **Processing:** Data is cleaned and converted  
3. **Model:** Machine learning algorithm processes input  
4. **Output:** Predicted disease  

---

## ✨ 7. Features
- User-friendly symptom input  
- Fast disease prediction  
- Uses severity-based weighting  
- Supports multiple symptoms  

---

## ✅ 8. Advantages
- Helps in early detection  
- Saves time  
- Easy to use  
- Supports healthcare decision-making  

---

## ⚠️ 9. Limitations

- Depends on dataset quality  
- Cannot replace professional medical advice  

---

## 🚀 10. Future Enhancements

## 📝 11. Conclusion
The Disease Prediction System demonstrates how machine learning can be used in healthcare to assist users in identifying potential illnesses based on symptoms.
